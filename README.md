# React Testing Practice Project

A beginner-friendly React project built with **React + TypeScript** to learn **component testing** using **Vitest** and **React Testing Library**.

Focus: testing **what the user sees and interacts with**, not internal state.

![React](https://img.shields.io/badge/React-17.0-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-4.9-blue)
![Vitest](https://img.shields.io/badge/Test-Vitest-brightgreen)

---

## Tech Stack
```
- React
- TypeScript
- Vite
- Vitest
- React Testing Library
- Bootstrap
```

---

## Components
- **Greet** – shows default and custom greeting messages  
- **Counter** – increment and decrement counter  
- **ThemeToggle** – toggle between light and dark themes  

Each component includes basic user-focused tests.

---

## Testing
- Tests focus on **UI output and user interactions**  
- Uses **`userEvent`** for simulating user actions  
- Avoids testing internal state
